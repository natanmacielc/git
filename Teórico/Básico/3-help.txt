Help

Se você precisar de ajuda para usar o Git, há três formas de acessar a página do manual de ajuda (manpage) para qualquer
um dos comandos Git:
$ git help <verb>
$ git <verb> --help
$ man git-<verb>

Por exemplo, você pode ver a manpage do commando config rodando:
$ git help config

Estes comandos podem ser acessados de qualquer lugar, mesmo desconectado. Se as manpages e este livro não forem suficientes e você
precisar de ajuda personalizada, você pode tentar os canais #git ou #github no servidor IRC Freenode (irc.freenode.net). 
Estes canais estão sempre cheios com centenas de pessoas que são bem versadas com o Git e dispostas a ajudar.
